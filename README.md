# Nostr Library 🟣

Library for Nostr

**Contributions are welcome**


## What is Nostr?

“nostr” stands for “Notes and Other Stuff Transmitted by Relays” and is an open protocol for censorship-resistant global networks created by @fiatjaf.

## Languages supported

- Javascript
- Swift
- Dart
- Flutter
- Typescript
- Rust
